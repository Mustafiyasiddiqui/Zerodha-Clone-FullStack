📈 Zerodha Clone (Full Stack Trading Platform)

A full-stack clone of **Zerodha**, a popular online stock trading platform.  
This project replicates the core features of Zerodha such as user authentication, live market data display, portfolio tracking, and modern responsive UI — built for learning and showcasing full-stack development skills.

---

🚀 Features

- 🧑‍💻 User authentication (Sign Up / Login)
- 📊 Real-time stock data display
- 💼 Portfolio and holdings management
- 💰 Buy/Sell simulation
- 🌙 Responsive and clean UI
- ⚡ Fast and dynamic React frontend
- 🗄️ RESTful backend API with Node.js + Express
- 💾 MongoDB database integration
